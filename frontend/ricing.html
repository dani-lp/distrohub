What is "ricing"?

"Rice" is the term used to refer to visual improvements and other customizations applied to a Unix/Linux system, shiny and good looking desktops that really catch your atention. However, getting into the world of ricing can seem daunting at first. The purpose of this section is to help begginer users to understand terms such as Window Managers, Desktop Environments, Terminal Emulators, Shells and so on.

❯ Desktop Environments

Most PC users use a desktop environment. They are a collection of program bundled together to make a workspace, and they all come with a window manager, a panel, a file manager, a terminal emulator, a set of default programs (text editors, music players, etc) and so on. For example, both Windows and macOS use a desktop environment. In Linux, some of the most used ones are:

  • Gnome
  • KDE
  • XFCE

❯ Tiling Window Managers

A window manager is software that manages the placement of open programs/windows. Most desktop environments have a window manager of their own; however, in the Linux community tiling window managers are very popular as well. They organize windows into mutually non-overlapping frames, often called tiles, so that every open program can be seen at all times. Some of the most popular ones are:

  • i3wm
  • bspwm
  • AwesomeWM

❯ Program Launchers

A launcher is a program that displays a dialog to perform an action, tipically launch an application. A lot of them also offer other functions, such as switch between windows or open files. They are often used in conjunction with tiling window managers. Some commonly used ones are:

  • Rofi
  • Dmenu

❯ Terminal Emulators

A terminal emulator is a program that emulates a video terminal within some other display architecture; most desktop environments come packed with their own. Often synonymous of with the term shell, the term terminal covers all remote terminals, including graphical interfaces. Some of them are:

  • Alacritty
  • Terminator
  • urxvt

❯ Shells

Shells are interactive programs to access the operating system's services. They are the typical command prompt that can often be seen in tv shows and movies with someone furiously typing nonsense. Using their commands, we can perform tasks such as file manipulation, user management, downloading software, and so on. There are a lot of command-line utilities that you can use as well. The most popular shells are:

  • Bash
  • Zsh
  • Fish

❯ System Info

Most fancy screenshots in /r/unixporn show their system specifications and their distro logo in a terminal. There are some command-line programs to do that; most of them are highly customizable and allow you to easily change the displayed information. The most popular ones are:

  • Neofetch
  • Screenfetch
